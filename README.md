# illusions-ruleset-jtf-style-guide

> illusions の校正ルールセット — **JTF日本語標準スタイルガイド**（日本翻訳連盟）

翻訳文・実務文書で求められる、表記・句読点・記号・単位の統一ルールを、小説エディタ
[illusions](https://illusions.app/) の自動校正に取り入れたものです。句読点（。、）への統一、
和文中の英数字の半角化、SI 単位記号の正しい大文字小文字、かっこ・スペースの約束事など、
**読み手に正確に伝えるための表記の標準化**をそろえます。

## このルールセットでできること

実務文書では、書き手によって表記がばらつくと読み手が混乱します。本ルールセットは、翻訳・実務
日本語で標準化された表記ルールに沿って、句読点の統一、半角カタカナの是正、全角英数字の半角化、
SI 単位記号の点検、半角・全角間スペースやかっこ周りのスペースの是正などをチェックします。
小説よりも、**マニュアル・解説・ビジネス文書・翻訳文**を書くときに力を発揮します。

## 収録ルール（21 件）

| ルール | 内容 | 例 |
| --- | --- | --- |
| 句読点の統一 | 句読点は全角「、」「。」 | `これは，例です．` → `これは、例です。` |
| 全角の読点・句点を使う | ピリオド・カンマを和文句読点に使わない | `終了し，保存．` → `終了し、保存。` |
| カタカナは全角で表記 | 半角カタカナを使わない | `ﾒｰﾙ` → `メール` |
| 算用数字の表記統一 | 全角数字を半角へ | `２５６ＭＢ` → `256MB` |
| 英数字は半角で表記 | 和文中の英数字は半角 | `ＡＢＣ` → `ABC` |
| 数字の桁区切りと小数点 | 桁区切りはカンマ、小数点はピリオド | `12，345．67` → `12,345.67` |
| 副詞などの漢字表記 | 特定の副詞は漢字で | `かならず` → `必ず` |
| 半角と全角の間のスペース | 半角と全角の間にスペースを入れない | `JTF 標準` → `JTF標準` |
| 句点の打ち方 | 文末の句点の有無を点検 | — |
| 閉じかっこ前の句点 | 閉じかっこの前に句点を打たない | `「発売します。」` → `「発売します」` |
| ピリオド・カンマを使わない | 和文の区切りに `.` `,` を使わない | `処理します.次へ` → `処理します。次へ` |
| かっこ周辺のスペース | かっこの内外にスペースを入れない | `クォータ （割当）` → `クォータ（割当）` |
| かっこは全角で表記 | 和文に隣接する半角かっこを全角へ | `[ファイル]` → `［ファイル］` |
| 長さの単位（SI） | m・cm・mm・km の正しい表記 | `3 MM` → `3 mm` |
| 質量の単位（SI） | g・kg・t の正しい表記 | `60 Kg` → `60 kg` |
| 面積・体積の単位（SI） | m²・m³・L の正しい表記 | `75 m2` → `75 m²` |
| 電気の単位（SI） | V・A・W・Ω・Hz の正しい表記 | `100 v` → `100 V` |
| 温度の単位（℃） | 摂氏記号の正しい表記 | `25°C` → `25℃` |
| 周波数の単位（SI） | Hz・kHz・MHz・GHz の正しい表記 | `2.4 ghz` → `2.4 GHz` |
| 速度の単位（SI） | m/s・km/h の正しい表記 | `120 KM/H` → `120 km/h` |
| 伝送速度の単位 | bps・kbps・Mbps・Gbps の正しい表記 | `100 MBPS` → `100 Mbps` |

各ルールの詳細は [`docs/rules/`](./docs/rules/) を参照してください。

## 出典について — 『JTF日本語標準スタイルガイド』

『JTF日本語標準スタイルガイド』は、**一般社団法人日本翻訳連盟（JTF）**が策定・公開している、
翻訳・実務日本語の表記を統一するための標準スタイルガイドです。翻訳成果物をはじめとする実務
文書で、句読点・記号・単位・かっこ・スペースなどの表記をそろえ、**読み手に正確に伝わる文章**を
書くための基準として広く参照されています。

- 原典（PDF）: <https://www.jtf.jp/tips/styleguide>
- ライセンス: **CC BY 4.0**（クリエイティブ・コモンズ 表示 4.0 国際）

本ルールセットは、上記ガイドラインで定められた**表記の約束事（事実・規約）**を、illusions の
校正エンジン向けに再実装したものです。ルールコード・例文はすべてオリジナルで、各ルールの
`sourceReference` には書名と章節番号のみを記載しています。

## illusions への導入

illusions（デスクトップ版）は、公式ルールセットを起動時に自動ダウンロードします。設定の
「校正と文体」に本ルールセットが表示され、各ルールを個別に有効・無効にできます。
