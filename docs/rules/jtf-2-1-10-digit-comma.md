# jtf-2-1-10-digit-comma — 数字の桁区切りと小数点

- **ruleId**: `jtf-2-1-10-digit-comma`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

算用数字の位取りの表記について、桁区切りには「カンマ」、小数点には「ピリオド」を使います。

## 出典

JTF日本語標準スタイルガイド 2.1.10 算用数字の位取りの表記

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `12,345.67` | 指摘なし |
| 誤例 | `12，345．67` | 指摘あり |
