# jtf-2-3-no-space — 半角と全角の間のスペース

- **ruleId**: `jtf-2-3-no-space`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

半角文字と全角文字の間に半角スペースを入れない。

## 出典

JTF日本語標準スタイルガイド 2.3 文字間のスペース

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `JTF標準` | 指摘なし |
| 誤例 | `JTF 標準` | 指摘あり |
