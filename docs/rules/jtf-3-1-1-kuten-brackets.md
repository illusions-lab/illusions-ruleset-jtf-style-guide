# jtf-3-1-1-kuten-brackets — 閉じかっこ前の句点

- **ruleId**: `jtf-3-1-1-kuten-brackets`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

文中にかぎかっこや丸かっこが入る場合は、閉じかっこの前に句点（。）を打ちません。

## 出典

JTF日本語標準スタイルガイド 3.1.1 句点（。）

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `A氏は「5月に新製品を発売します」と述べました。` | 指摘なし |
| 誤例 | `A氏は「5月に新製品を発売します。」と述べました。` | 指摘あり |
