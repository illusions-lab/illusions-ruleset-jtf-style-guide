# jtf-2-2-1-kanji — 副詞などの漢字表記

- **ruleId**: `jtf-2-2-1-kanji`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

特定の副詞などは、ひらがなではなく「漢字」で表記します。

## 出典

JTF日本語標準スタイルガイド 2.2.1 ひらがなと漢字の使い分け

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `必ず一切の責任を負う。` | 指摘なし |
| 誤例 | `かならずいっさいの責任を負う。` | 指摘あり |
