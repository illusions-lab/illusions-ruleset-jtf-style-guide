# jtf-2-1-8-halfwidth-alnum — 英数字は半角で表記

- **ruleId**: `jtf-2-1-8-halfwidth-alnum`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

算用数字とアルファベットは半角で表記します（横組みの場合）。全角の英数字は使用しません。

## 出典

JTF日本語標準スタイルガイド 2.1.8 算用数字、2.1.9 アルファベット

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `12345、abc DEF` | 指摘なし |
| 誤例 | `１２３４５、ａｂｃ ＤＥＦ` | 指摘あり |
