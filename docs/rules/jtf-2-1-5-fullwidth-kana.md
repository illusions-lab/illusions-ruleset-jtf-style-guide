# jtf-2-1-5-fullwidth-kana — カタカナは全角で表記

- **ruleId**: `jtf-2-1-5-fullwidth-kana`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

漢字、ひらがな、カタカナは全角で表記します。半角カタカナは使用しません。

## 出典

JTF日本語標準スタイルガイド 2.1.5 カタカナ

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `メールアドレス` | 指摘なし |
| 誤例 | `ﾒｰﾙｱﾄﾞﾚｽ` | 指摘あり |
