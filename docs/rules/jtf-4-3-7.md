# jtf-4-3-7 — 周波数の単位（SI）

- **ruleId**: `jtf-4-3-7`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

周波数について、SI単位（Hz、kHz、MHz、GHz）を正しく表記します。

## 出典

JTF日本語標準スタイルガイド 4.3.7 周波数

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `50 Hz、2.4 GHz` | 指摘なし |
| 誤例 | `50 hz、2.4 ghz` | 指摘あり |
