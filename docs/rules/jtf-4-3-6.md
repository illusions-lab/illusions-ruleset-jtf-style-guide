# jtf-4-3-6 — 温度の単位（℃）

- **ruleId**: `jtf-4-3-6`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

温度について、摂氏（℃）を正しく表記します。

## 出典

JTF日本語標準スタイルガイド 4.3.6 温度

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `室温 25℃` | 指摘なし |
| 誤例 | `室温 25°C` | 指摘あり |
