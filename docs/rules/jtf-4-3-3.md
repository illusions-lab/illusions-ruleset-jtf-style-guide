# jtf-4-3-3 — 質量の単位（SI）

- **ruleId**: `jtf-4-3-3`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

質量について、SI単位（g、kg、t）を正しく表記します。

## 出典

JTF日本語標準スタイルガイド 4.3.3 質量

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `体重 60 kg、荷物 500 g` | 指摘なし |
| 誤例 | `体重 60 Kg` | 指摘あり |
