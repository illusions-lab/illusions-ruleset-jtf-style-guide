# jtf-4-3-2 — 長さの単位（SI）

- **ruleId**: `jtf-4-3-2`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

長さについて、SI単位（m、cm、mm、km）を正しく表記します。

## 出典

JTF日本語標準スタイルガイド 4.3.2 長さ

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `全長 1.5 m、厚さ 3 mm` | 指摘なし |
| 誤例 | `全長 1.5 M、厚さ 3 MM` | 指摘あり |
