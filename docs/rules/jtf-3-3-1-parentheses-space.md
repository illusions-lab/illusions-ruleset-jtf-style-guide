# jtf-3-3-1-parentheses-space — かっこ周辺のスペース

- **ruleId**: `jtf-3-3-1-parentheses-space`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

かっこの外側、内側ともにスペースを入れません。

## 出典

JTF日本語標準スタイルガイド 2.3.2 かっこ類と隣接する文字の間のスペースの有無

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `クォータ` | 指摘なし |
| 誤例 | `クォータ` | 指摘あり |
