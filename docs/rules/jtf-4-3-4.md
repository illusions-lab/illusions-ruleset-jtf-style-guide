# jtf-4-3-4 — 面積・体積の単位（SI）

- **ruleId**: `jtf-4-3-4`
- **ガイドライン**: JTF日本語標準スタイルガイド (`jtf-style-guide`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

面積、体積について、SI単位（m²、m³、L）を正しく表記します。

## 出典

JTF日本語標準スタイルガイド 4.3.4 面積、体積

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `床面積 75 m²、容量 500 mL` | 指摘なし |
| 誤例 | `床面積 75 m2` | 指摘あり |
